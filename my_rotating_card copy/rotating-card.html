Database Design

During the summer of 2015, I worked with the FAA’s tool, ERAM (Enroute Automation Modernization) network. I built a new database containing output from vulnerability scans of the network and worked with users to determine requirements for the new database. I then designed and built the new database using Microsoft Access and SQL to compile system vulnerability scan results, analyze the data, and generate reports. The new system replaced the time-consuming manual process of manipulating Excel spreadsheets and reduced the process time by 93%.
